CSS Structure

There are 6 main CSS files in this theme.


If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.


Open The css Folder, Find All CSS Files.

  • animate.css - Animation Style
  • bootstrap.min.css - Bootstrap Style
  • font-awesome.min.css - Font Awesome Style
  • slick.css - Slick Style
  • slick-theme.css - Slick Theme Style
  • style.css - Theme Common Styles With Responsive Style In It

JAVAScript Structure

This theme imports 7 Javascript files. All located in the "js" folder.


Open The js Folder, Find All JAVA Script Files.

  • bootstrap.min.js - Bootstrap Javascript
  • custom.js - Main Custom Javascript
  • fancybox.js - FancyBox Javascript
  • anime.js - Anime Javascript
  • jquery-3.5.1.min.js - JQuery v3.5.1 Javascript
  • slick.min.js - Slick Javascript
  • wow.min.js - WOW Javascript
